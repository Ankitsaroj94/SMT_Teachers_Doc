# Skill: figma-generate-library

Build/update design-system libraries in Figma.

## Preferred tools

- `plugin-figma-figma/create_design_system_rules`
- `plugin-figma-figma/use_figma`
- `plugin-figma-figma/search_design_system`

## Fallback on user MCP

- User MCP currently supports read/image flows only.
- If plugin server is unavailable, generate a detailed library plan and token spec in docs, then apply once plugin access is restored.

## Delivery checklist

- Token set coverage (color/spacing/type/radius/elevation)
- Component inventory and variants
- Naming conventions and usage docs
